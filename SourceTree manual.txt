	1. 클론을 이용한 방법 (새로운 프로젝트, 기존 프로젝트 모두 가능)
		1. Github에 리포지토리 생성
		2. URL 복사
		3. SourceTree 클론 생성
			소스경로 : Github의 복사한 URL
			목적지경로 : 반드시 빈 폴더로 해야함
			이름 : 소스트리에 저장할 프로젝트 이름
		4. 목적지경로로 설정한 빈 폴더에 프로젝트 소스 넣기
		5. 리포지토리 - 파일상태 - 커밋
		
		
		
	2. 로컬 리포지토리 생성하는 방법 (새로운 프로젝트, 기존 프로젝트 모두 가능)
		1. SourceTree Create 
			목적지경로 : 폴더
			이름 : 소스트리에 저장할 프로젝트 이름
		2. 계정에 저장소 생성하기 (Bit_buket이 아니라면 미체크)
		3. 생성
		4. 해당 리포지토리 탭을 클릭한 상태에서 
			저장소 - 저장소설정 - 원격 - 추가 - 원격이름(저장할 이름) - URL 경로(Github URL) - Remote Account(연결할 Github설정)
		5. 만들어진 원격저장소 더블클릭
		6. 리포지토리 - 파일상태 - 커밋
